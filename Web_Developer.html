UWaterloo VIP Lab / Jan 2023 – Apr 2023

Responsibilities

Website Design

At the Vision and Image Processing Lab, I had two main tasks. The first involved the full development of a completely automated research lab website (vip.uwaterloo.ca) with user account functionality. I had to make this website with no prior web development experience. I was given numerous requirements to be met, including various forms of automation on the site. I met these requirements using API integration, a PHP backend, Wordpress, and significant frontend coding.

SAR Data Acquisition

The second task I worked on was assisting research by performing data acquisition and analysis of synthetic aperture radar (SAR) images of Northern ice. This was to produce valid and effective data for a research project relating to the eider duck and how changing volumes of ice impacted its survival and migration patterns. I worked with Matlab software which automatically segmented the images into ice, land, and water. I would then go through and manually make changes where I felt the software was incorrect.
